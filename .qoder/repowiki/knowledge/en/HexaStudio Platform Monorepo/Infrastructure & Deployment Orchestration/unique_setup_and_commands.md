`scripts/setup-server.sh` must be run as root on a fresh Ubuntu 24.04 host before deployment (configures fail2ban, UFW, disables root SSH login, enables unattended upgrades). Production secrets are generated with `bash scripts/generate-secrets.sh` which writes `.env.production`; deploy with `bash scripts/deploy.sh deploy` (rolling update with health checks) or `bash scripts/deploy.sh db-migrate` / `db-seed`.

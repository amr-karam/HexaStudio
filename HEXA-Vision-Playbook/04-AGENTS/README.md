## Agent Communication Protocol

All agents must communicate via:
1. **GitHub Issues** — Feature requests, bug reports, tasks
2. **Pull Requests** — Code changes with descriptions
3. **ADR Documents** — Architectural decisions
4. **Meeting Notes** — Synchronous discussion records

## Agent Verification

Every agent must self-verify against the agent checklist in `AI_AGENT_GUIDE.md` before completing any task.

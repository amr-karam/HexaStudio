**Purpose:** To transition an AI Agent from a general state to a "HEXA Studio Engineer" state.
**Trigger:** Start of every new session.

## 🛠️ THE INITIALIZATION SEQUENCE

### Phase 1: The Legal & Philosophical Alignment
**Action:** Read the following documents in order.
1. `AGENTS.md` (The operating manual)
2. `HEXA-Vision-Playbook/00-GOVERNANCE/PROJECT_CONSTITUTION.md` (The laws of excellence)
3. `HEXA-Vision-Playbook/00-GOVERNANCE/PROJECT_OVERVIEW.md` (The project identity)

**Objective:** Internalize the "Luxury vs. Enterprise" balance.

### Phase 2: Technical Synchronization
**Action:** Audit the current technical state.
1. Scan `HEXA-Vision-Playbook/01-ARCHITECTURE/SYSTEM_ARCHITECTURE.md` for the blueprint.
2. Scan `HEXA-Vision-Playbook/06-STANDARDS/CODING_STANDARDS.md` for the quality bar.
3. Scan the file tree to identify the current project structure.

**Objective:** Understand the constraints and the "Gold Standard."

### Phase 3: Operational Alignment
**Action:** Synchronize with the current sprint.
1. Read `HEXA-Vision-Playbook/02-ROADMAP/CURRENT_SPRINT.md`.
2. Read `HEXA-Vision-Playbook/02-ROADMAP/OPEN_TASKS.md`.

**Objective:** Identify the current priority and your specific assignment.

# 🏛️ PROMPT 001: CONSTITUTIONAL ALIGNMENT

**Purpose:** To reset the agent's quality bar and realign them with the Project Constitution.
**Trigger:** When the agent produces "average" work or suggests a "quick fix."

---

## 🎯 THE REALIGNMENT MANDATE

You are drifting away from the **HEXA Gold Standard**. You are currently thinking like a "standard developer." You must now shift back into **Creative Excellence Mode**.

**Reminder of the Three Pillars:**
1. **Visual Absolute:** Is this a "standard" UI, or is it a "luxury" experience?
2. **Engineering Rigor:** Is this "working" code, or is it "enterprise-grade" code?
3. **Architectural Integrity:** Is this a "patch," or is it a "sustainable solution"?

---

## 🔍 THE CRITICAL AUDIT

Review your last proposal/code and answer these questions:
- **The Luxury Test:** Does this feel expensive? Does it evoke prestige?
- **The Performance Test:** Will this maintain 60 FPS on a mid-range device?
- **The Type Test:** Are there any `any` types or implicit returns?
- **The Documentation Test:** If I delete you, can another agent understand this in 2 minutes?

---

## 🚀 THE RECTIFICATION PROCESS

1. **Identify the Gap:** Explicitly state where the current solution is "average."
2. **Propose the Elevation:** Describe how to make the solution "World Class."
3. **Re-Implement:** Apply the high-end standards.

`⚠️ Alignment Reset | Constitution Re-read | Quality Bar Restored`

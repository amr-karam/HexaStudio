# Checklists Directory

This directory contains verification checklists for the HEXA Vision platform.

## Contents

| File | Description |
|------|-------------|
| `pre-commit-checklist.md` | Checklist before every commit |
| `pre-pr-checklist.md` | Checklist before opening a PR |
| `pre-release-checklist.md` | Checklist before every release |
| `accessibility-checklist.md` | WCAG 2.1 AA verification |
| `performance-checklist.md` | Performance budget verification |
| `security-checklist.md` | Security review checklist |
| `3d-checklist.md` | 3D scene quality checklist |
| `responsive-checklist.md` | Responsive design verification |
| `seo-checklist.md` | SEO verification |

## Usage

Run through the appropriate checklist at each stage of development. If any item fails, fix before proceeding.

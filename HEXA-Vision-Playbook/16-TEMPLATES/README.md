# Templates Directory

This directory contains reusable document templates for the HEXA Vision project.

## Contents

| File | Description |
|------|-------------|
| `TEMPLATE_Feature_Story.md` | Template for feature implementation stories |
| `TEMPLATE_PR.md` | Template for Pull Request descriptions |
| `TEMPLATE_Meeting_Notes.md` | Template for meeting records |
| `TEMPLATE_Sprint_Review.md` | Template for sprint retrospectives |
| `TEMPLATE_Incident_Report.md` | Template for post-incident reports |
| `TEMPLATE_Bug_Report.md` | Template for bug reports |
| `TEMPLATE_Release_Notes.md` | Template for release notes |

## Usage

Copy the relevant template and fill in the details. Maintain the section structure but adapt content as needed.

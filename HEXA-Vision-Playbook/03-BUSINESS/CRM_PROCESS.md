# 🤝 CRM PROCESS: THE RELATIONSHIP ENGINE

**Version:** 1.0 | **Domain:** Business | **Focus:** Client Lifecycle Management

## 1. THE CRM PHILOSOPHY
CRM at HEXA Studio is not about "managing leads"; it is about **Nurturing Partnerships**. Every interaction with a client must feel as premium as the website itself.

---

## 2. THE CLIENT LIFECYCLE

### Stage 1: Lead Intake (The "Greeting")
- **Action:** Lead submits a request.
- **System:** Data flows from Website $\rightarrow$ NestJS $\rightarrow$ Odoo CRM.
- **Touchpoint:** An immediate, sophisticated confirmation email.

### Stage 2: Qualification (The "Filter")
- **Action:** Discovery call with the Project Manager.
- **System:** Log call notes and qualification scores in Odoo.
- **Goal:** Determine if the project aligns with the "Luxury" standard.

### Stage 3: Proposal (The "Vision")
- **Action:** Create a custom project proposal with a visual moodboard.
- **System:** Generate proposal document $\rightarrow$ Send via Client Portal.
- **Goal:** Secure commitment to the vision and budget.

### Stage 4: Execution (The "Journey")
- **Action:** Weekly updates, milestone approvals.
- **System:** Project tracking in Odoo $\rightarrow$ Client Portal.
- **Goal:** Maintain transparency and trust.

### Stage 5: Handover (The "Masterpiece")
- **Action:** Final delivery and training.
- **System:** Archive project files $\rightarrow$ Request testimonial.
- **Goal:** Transition to long-term maintenance.

---

## 3. ODOO INTEGRATION POINTS
- **Leads $\rightarrow$ Opportunities:** Automatic conversion based on qualification.
- **Projects $\rightarrow$ Tasks:** Syncing Odoo project tasks with the internal implementation roadmap.
- **Invoices $\rightarrow$ Payments:** Automated billing based on milestone completion.

---

## 4. COMMUNICATION STANDARDS
- **Tone:** Professional, confident, and supportive.
- **Frequency:** Weekly updates for active projects; monthly check-ins for dormant leads.
- **Channel:** Email for formal records; Slack/WhatsApp for rapid coordination.

*“The relationship is the real product; the website is just the evidence.”*
